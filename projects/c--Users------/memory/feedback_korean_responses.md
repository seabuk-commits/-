**How to apply:** 이후 모든 세션에서 설명, 안내, 요약 등 응답 텍스트는 한국어로 작성한다. Git, PowerShell, GitHub, NotebookLM 같은 고유 명사/제품명이나 명령어 자체는 번역하지 않고 그대로 둔다.
