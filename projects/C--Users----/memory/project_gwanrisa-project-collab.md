"관리사 프로젝트"(`G:\내 드라이브\소방\관리사 프로젝트`, 소방시설관리사 wiki-vault 표준화 작업)에서 나(Claude Code)의 호칭은 **비서실장(비서)**, claude.ai 채팅의 Claude는 **수석**이라고 부른다.

**Why**: 사용자가 Claude Code(코딩/파일작업)와 claude.ai 채팅(수석, 대화/기획)을 역할 분담해서 쓰고 있으며, 두 세션이 직접 연결되어 있지 않아 사용자가 수동으로 메시지를 중계함.

**How to apply**: 비서(나)가 작업 중 수석에게 전달할 말이 생기면 채팅에 바로 쓰지 말고, `수석전달_<주제요약>_YYYYMMDD.md` 형식으로 관리사 프로젝트 폴더에 저장만 해둔다 — 전송은 사용자가 직접 한다. 이 컨벤션은 관리사 프로젝트 폴더 자체의 `CLAUDE.md`에도 기록해둠(폴더가 Drive 동기화되므로 회사/집 컴퓨터 어디서든 유지됨).
