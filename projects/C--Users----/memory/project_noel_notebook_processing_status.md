**목표**: 노엘(NotebookLM) 노트북에 쌓인 소스를 전부 가공(암기노트/리포트/퀴즈/플래시카드 등 산출물로
전환)할 때까지 계속 진행. 가공 완료된 원본 소스는 [[노엘]] 정책에 따라 삭제 대상 목록을 구체적으로
나열해 확인받은 뒤 삭제(회전형 워크플로, 한도 관리 A안).

**Why**: 기술사님이 "노엘이 자료 다 없어질 때까지 진행"이라고 명시적으로 지시함(2026-07-16). 또한
소방기사(소방설비산업기사) 자료는 5개 자격증 트랙 중 가장 기초 자료라 관리사·기술사 등 상위 자격증
자료와 내용이 많이 겹친다고 지적함 — 다른 노트북 처리 시 이 중복성을 참고해서 정리(중복 내용은
간소화하거나 상호 참조 표시 등으로 처리, 맹목적으로 매번 새로 만들지 말 것).

**How to apply**:
- 노트북 5개: 관리사(소방시설관리사), 전기기사(소방설비기사-전기), 관리사추가(소방시설관리사-추가자료),
  기술사(소방기술사), 기계기사(소방설비기사-기계). UUID는 [[project_notebooklm_full_roadmap_state]] 참고
  (매번 `notebook_list`로 재확인 — 기록에 의존 금지).
- **전기기사 노트북은 2026-07-15~16 세션에서 처리 완료**: Study Guide 리포트, 퀴즈, 플래시카드,
  마인드맵을 노엘 studio_create로 신규 생성 + 기존에 있던 리포트 2종(단답 250제/250선)·시설설계
  가이드라인·퀴즈 1종도 회수. 전부 `관리사 프로젝트/산출물/소방기사전기_암기노트/A~L` 파일로 저장,
  200줄 넘는 파일은 표 형식으로 압축(내용 손실 없이 포맷만 압축, 스크립트로 기계적 변환). 가공 완료된
  원본 PDF 소스 12개는 노트북에서 삭제(50→37).
- **남은 작업**: 관리사, 관리사추가, 기술사, 기계기사 4개 노트북에 대해 동일한 가공(리포트/퀴즈/
  플래시카드/마인드맵 생성 → 로컬 다운로드 → 압축 저장 → 가공된 원본 소스는 확인 후 삭제) 반복.
  각 노트북 처리 시작 전 소방기사(전기기사) 자료와 겹치는 내용이 있는지 먼저 확인.
